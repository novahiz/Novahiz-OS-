# Workflow: Data Analysis

> Standard protocol for exploring and analyzing data.

## Trigger
- Data exploration request
- `analyze`, `data`, `insights`, `report`, `xlsx`, `csv`

## Steps

### 1. Scope (Maya + Elias)
- [ ] What data source? (DB, API, file)
- [ ] What question are we answering?
- [ ] Output format? (report, chart, summary)

### 2. Extract (Maya)
- [ ] Pull data with xlsx skill
- [ ] Clean anomalies
- [ ] Validate completeness

### 3. Analyze (Maya + Elias)
- [ ] Statistical summary
- [ ] Trends over time
- [ ] Segments / cohorts
- [ ] Anomalies / outliers

### 4. Visualize (Luna)
- [ ] Pick right chart type
- [ ] Keep it simple
- [ ] Actionable labels

### 5. Deliver (Elias)
- [ ] Write executive summary
- [ ] Include methodology
- [ ] Next steps / recommendations
