## 📝 Installation Commands (for README)

### Linux/macOS
```bash

```

### Windows (PowerShell)
```powershell
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/novahiz/novahiz-os/main/install.ps1" -OutFile "install.ps1"
.\install.ps1
```

cd novahiz-os
python3 install.py
