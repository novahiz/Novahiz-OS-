⚠️ Requirements

🔄 OpenCode Only

Novahiz OS runs exclusively on the OpenCode framework. API keys for OpenRouter or other providers required for agent execution.

🐧 WSL Required on Windows

Native Windows is not supported. Install WSL first: wsl --install then run the Linux installer inside WSL.

Install

🐧 Linux / macOS

curl -fsSL https://raw.githubusercontent.com/novahiz/Novahiz-OS-/main/install.sh | bash

🪟 Windows (WSL Required)

powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((Invoke-RestMethod 'https://raw.githubusercontent.com/novahiz/Novahiz-OS-/main/install.ps1'))"

Installs WSL + Novahiz OS automatically
