## 3. Trademark Usage Guidelines

### 3.1 Correct Usage

**✅ DO:**

- Use ™ symbol on first mention: **Novahiz OS™**
- Use as adjective + noun: "Novahiz OS software"
- Distinguish from surrounding text (bold, caps, etc.)
- Include trademark notice in documentation

**Examples:**
- ✅ "Novahiz OS™ is an orchestration system"
- ✅ "The Novahiz OS™ platform"
- ✅ "Powered by Novahiz OS™"

### 3.2 Incorrect Usage

**❌ DON'T:**

- Use as verb: "Novahiz this task"
- Use as possessive: "Novahiz's features"
- Pluralize: "Novahiz OSes"
- Abbreviate: "Novahiz" (without OS)
- Combine with other marks
